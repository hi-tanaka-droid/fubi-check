# フビチェック

「学会参加補助制度」(産業社会学会・院生委員会)の申請書類を、写真からAIがチェックするアプリです。

**アプリを開く → https://hi-tanaka-droid.github.io/fubi-check/**

![QRコード](qr.png)

## 使い方

1. 上のQRコードをスマホで読み取ってアプリを開く
2. 申請書・領収書・プログラム・通帳コピーなどの写真を追加
3. 「チェックする」を押すと、募集要項と過去の頻出ミスに照らした点検結果が「要修正/要確認/OK」で表示されます

初回のみ、右上の ⚙︎ からAnthropic APIキー(sk-ant-…)の設定が必要です。

## ホーム画面に追加(アイコン設置)

- **iPhone (Safari)**: 共有ボタン(□↑) → 「ホーム画面に追加」
- **Android (Chrome)**: メニュー(⋮) → 「ホーム画面に追加」または「アプリをインストール」

以後、ホーム画面のアイコンからアプリとして起動できます。

## プライバシー

- 写真はチェック時にAnthropic APIへ暗号化通信で送られるだけで、このアプリ自体はどこにも保存しません
- APIキーは端末のブラウザ内にのみ保存されます
- 通信先は api.anthropic.com のみに制限しています(CSP)
